## Changes

### Added

- Added update checks in the app, with release notes and an update entry shown directly when a newer version is available.
- Added background notifications for completed chats, input requests, approvals, and failed runs.
- Knowledge documents and directories now refresh automatically after files change on disk.
- The Unity plugin now installs to `Packages/com.farlocus.locus` by default and can migrate existing installs from the old locations.

### Fixed

- Improved version comparison and release metadata validation so the app version, update manifest, and release notes stay aligned.
- Refined session TODO and transcript behavior so the current run scope is clearer and tool calls render in a more stable order.
- Updated the Chinese and English docs and install guide to match the current plugin path and product behavior.

### Removed

- Removed the old default Unity plugin install flow for `Assets/Locus` and `Assets/Plugins/Locus`.
